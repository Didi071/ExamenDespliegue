El proyecto este estrucurado en la siguiente estructura

Examne Despliegue es la carpeta principal. Dividida en las siguientes subcarpetas:

1. docs (contiene toda la documentacion con el readme, arquitecuta.md, guia-instalación entre otros)
2. node_modules (todas las aplicaciones con jsdocs o jest)
3. src (el proyecto index.html, style.css y app.js)
4. test (el test de app.js llamado app.test.js)

Y los siguientes archivos

1. .gitignore -> Todo lo que se debe igonrar como logs y node modueles
2. index.html -> Pagina de redireccion a src/index.html
3. jsdoc.json -> Archivo de configuracion para generar documentacion
4. package-lock.json -> Dependencias
5. package.json -> Toda la configuracion del proyecto en node
6. README.md -> Descripción del proyecto
